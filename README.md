# Developer CV Template

## Tech Stack
- React
- SCSS

## How to Use
0) npm install
1) npm start
2) Modify data in public/cv.json
3) Replace public/headshot.png
4) Change Title in src/index.html
5) Print to PDF with Google Chrome
